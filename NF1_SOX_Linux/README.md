## Introducció als servidors GNU/Linux

En el món dels servidors, GNU/Linux, a diferència dels entorns d'escriptori, és el sistema operatiu més utilitzat. La seva robustesa, seguretat i flexibilitat el fan ideal per a entorns de producció.

- Servidors Web (Rànking Global): A prop del 96,3% del milió de llocs web principals usen Linux.
- Servidors Globals: Linux està instal·lat en aproximadament el 53% de tots els servidors físics i virtuals del món.
- La totalitat dels superordinadors més potents del món utilitzen Linux com a sistema operatiu.
